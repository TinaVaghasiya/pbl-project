# Sign Language Images

This directory contains sign language gesture images.

## Image Naming Convention

- Alphabets: `a.png`, `b.png`, ..., `z.png`
- Numbers: `0.png`, `1.png`, ..., `9.png`
- Words: `hello.png`, `thanks.png`, etc.

## Image Requirements

- Format: PNG or JPG
- Size: 512x512 pixels (recommended)
- Background: Transparent or white
- Clear hand gesture visibility

## Adding Images

1. Place image files in this directory
2. Update the database seed file if needed
3. Ensure filenames match the database entries

## Sample Images

You can download ASL (American Sign Language) alphabet and number images from:
- https://www.signingsavvy.com/
- https://www.lifeprint.com/
- Create your own using the webcam feature

## Note

For demo purposes, the application will display letter/number placeholders if images are not available.
